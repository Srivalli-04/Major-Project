# Real-Time Disaster Alert and Assisting Management System

## Overview

The Real-Time Disaster Alert and Assisting Management System is a web-based application designed to provide real-time disaster alerts and support emergency response management. The system helps users and authorities monitor disasters, track affected areas, and allocate resources efficiently during emergencies.

## Problem Statement

Natural disasters such as floods, earthquakes, fires, and storms often result in delayed communication and inefficient resource allocation. This project aims to improve disaster response by providing timely alerts and assisting decision-makers with resource management.

## Objectives

* Provide real-time disaster alerts.
* Enable incident reporting through a user-friendly interface.
* Assist in emergency resource allocation.
* Visualize disaster information through dashboards and alerts.
* Improve disaster response and monitoring.

## Features

* 🚨 Real-time disaster alerts
* 📍 Incident reporting system
* 📊 Interactive dashboard
* 🚑 Resource allocation estimation
* ⚡ Dynamic updates without page reload
* 🌐 Web-based accessibility

## Technologies Used

* HTML
* CSS
* JavaScript
* FastAPI
* Machine Learning Concepts
* IoT Concepts

## System Workflow

1. User/Admin reports a disaster incident.
2. The system processes the input details.
3. Resource requirements are estimated.
4. Real-time alerts and notifications are generated.
5. Results are displayed on the dashboard for monitoring and decision-making.
